درگاه پرداخت زرین‌پال برای EDD (نسخه REST API V4)

ZarinPal Payment Gateway for Easy Digital Downloads (V4)

این افزونه، نسخه بازنویسی‌شده و کاملاً به‌روزرسانی‌شده‌ی درگاه پرداخت زرین‌پال برای افزونه فروشگاه‌ساز Easy Digital Downloads (EDD) در وردپرس است.

با توجه به تغییرات ساختاری وب‌سرویس‌های زرین‌پال از اواخر سال ۱۴۰۴ و منسوخ شدن پروتکل‌های قدیمی (SOAP)، نسخه‌های قبلی این افزونه با خطاهایی مانند «رکورد مورد نظر یافت نشد» یا قطعی در اتصال مواجه می‌شدند. این مخزن برای رفع کامل این مشکلات و هماهنگی با مستندات جدید زرین‌پال توسعه یافته است.

🌟 ویژگی‌های نسخه جدید (نسخه 2.2.0)

مهاجرت کامل به REST API (نسخه 4): حذف وابستگی به SoapClient و استفاده از ساختار استاندارد JSON.

حل قطعی خطای بازگشت (Session): حذف وابستگی به $_SESSION مرورگر و استفاده از پارامترهای URL برای پیگیری دقیق تراکنش‌ها پس از بازگشت از بانک (سازگار با تمامی افزونه‌های کش).

ارسال هوشمند متادیتا: ارسال اتوماتیک شماره سفارش (Order ID) وردپرس، ایمیل و شماره موبایل خریدار به رسید زرین‌پال.

پشتیبانی هوشمند از ارزها: شناسایی خودکار واحد پولی تنظیم شده در EDD (تومان یا ریال) و تبدیل صحیح مبالغ.

کدنویسی تمیز و بهینه‌تر: حذف تنظیمات منسوخ شده (وب‌گیت و سرور آلمان) از پنل مدیریت.

📥 راهنمای نصب

ابتدا اگر نسخه قدیمی افزونه زرین‌پال EDD را دارید، آن را غیرفعال و حذف کنید.

فایل edd-zarinpal.php را از این مخزن دانلود کنید.

در مسیر wp-content/plugins/ هاست خود یک پوشه به نام edd-zarinpal بسازید و فایل را داخل آن قرار دهید (یا فایل را زیپ کرده و از طریق پیشخوان وردپرس آپلود کنید).

افزونه را در وردپرس فعال کنید.

به مسیر دانلودها > پیکربندی > درگاه‌های پرداخت بروید، درگاه زرین‌پال را فعال کرده و مرچنت‌کد (Merchant ID) خود را وارد کنید.
